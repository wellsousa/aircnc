O backend deverá rodar durante todo o desenvolvimento da aplicação frontend.

OU 

npm run dev 

- execute o comando para start o servidor do frontend, por exemplo, no diretorio

yarn start

OU 

npm start 

A aplicação será iniciada na porta 3000 por padrão e carregada no browser definido como padrão no sistema.
